Final Project

My final project is a Chrome Extension that allows you to write a notes and post it to any website. All users with the extension can see messages from other users at that URL.

Click here to download

How to Install
  1. Download from the link above
  2. Go to your chrome extensions (go to chrome://extensions/)
  3. Drag the downloaded link into the chrome extensions page
How to Use
  1. Click on the extension icon by the address bar
  2. Type your message and hit enter or click on submit
  3. Your message will post to the website you are on for everyone to see
